# Price Prediction using Regression

## Objective
Accurately estimating property values is a complex task for real estate firms due to the high volume of independent variables that influence 
market prices. The objective is to develop a machine learning regression model that analyzes house features to predict the final sale price 
for each property ID in the dataset.

## Dataset
Source + brief description

## Approach
- Data Exploration
  - Checking data type
- Data Cleaning
  - Target log transformation 
  - Merging data sets
  - Performing missing value imputation 
  - Performing outlier analysis
  - Performing feature engineering
  - Performing data encoding
- Data Visualization 
 - Performed correlation to find the top 10 predictor-predictor correlations

## Predictive Analytics
- Used various regression algorithms such as Linear Regression, Lasso Regression & Random Forest Regression to predict the best house
  sale price.
-  Since Lasso Regression gives the lowest RMSE, it was used to predict the house sale price.

## Tools Used
Python, Pandas
